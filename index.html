Программа «Заказ на Маркете»

Программа «Заказ на Маркете» — это размещение предложений на Яндекс.Маркете по модели оплаты за действие (Cost per Action, CPA):

  1. Покупатели добавляют товары вашего магазина в корзину на Яндекс.Маркете и оформляют заказ прямо на сервисе.
  2. Оформленные заказы поступают в ваш магазин — вы платите только за готовые заказы.

Программа доступна для всех интернет-магазинов, размещающихся на Яндекс.Маркете.

Преимущества для магазина

При обычном размещении на Яндекс.Маркете магазин оплачивает все переходы на его сайт, однако лишь часть таких переходов конвертируется в покупку. Участвуя в программе «Заказ на Маркете», вы получаете не потенциального покупателя, а уже оформленный заказ. Конверсия в заказ происходит на стороне сервиса, и вы не теряете деньги.

Сколько это стоит?

За заказ с постоплатой магазин оплачивает комиссию 2% от стоимости заказа. На заказ с предоплатой Яндекс.Маркет предоставляет магазину скидку и не взимает с него комиссию за заказ. Магазин оплачивает только комиссию за проведение оплаты (2%).

Как начать пользоваться?

  1. На странице настроек размещения включить «Заказ на Маркете», указать режим приема заказов через API, подать заявку на прием предоплаты
  2. Включить модуль «Yandex.Market order»
  3. На странице настроек API заказа указать параметры для запросов от Маркета к магазину
  4. В админке модуля «Yandex.Market order» указать параметры для ответов/запросов магазина Маркету

Страница настроек API заказа. Немного технического языка :)

  • Авторизационный токен: токен, передываемый при каждом запросе от Маркета к магазину
  • URL API: URL, по которому Маркет отправляет запросы на создание/отмену заказа
  • SHA1 fingerprint: уникальный ключ для SSL-сертификата
  • Тип авторизации: будет ли передан авторизационный токен в заголовке или как пареметр запроса
  • Формат данных: в json или xml Маркет отправляет запросы на создание/отмену заказа

Админка модуля «Yandex.Market order»

  • ID приложения/авторизационный токен на Яндекс.OAuth
  • Кампания для подключения к «Заказу на Маркете»
  • Прайс-лист кампании
  • Имя службы доставки
  • Цена доставки
  • Длительность доставки
